
# 全球空气质量检测可视化

此项目基于云开发 + [Express](https://expressjs.com/zh-cn/) 展示全球空气质量检测页面

## 线上演示 & 一键部署！

点击下方按钮使用 [CloudBase Framework](https://github.com/TencentCloudBase/cloudbase-framework) 可以在云端一键部署本项目到自己的云开发账号上。

👇👇👇 点击一键部署自己的页面
